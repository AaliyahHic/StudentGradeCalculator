## Student Grade Calculator - README

### Overview
A beginner-friendly Java console application that calculates student grades using weighted scores. Supports multiple students and generates class statistics.

### Features
- Enter multiple students
- Validated input for all scores
- Weighted final grade calculation
- Letter grade conversion
- Class summary with average, highest, lowest grades

### How to Run
1. Save `StudentGradeCalculator.java` to your folder.
2. Open terminal in that folder.
3. Compile the program:
   javac StudentGradeCalculator.java
4. Run it:
   java StudentGradeCalculator
